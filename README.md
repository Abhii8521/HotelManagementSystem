Hotel Management System (Java)

A console-based Hotel Management System built in Java that automates room booking, food ordering, billing, and customer management.  
It demonstrates strong use of Object-Oriented Programming (OOP), File Handling, Serialization, and Multithreading concepts.

# Features
-  Room booking for single/double rooms (Luxury and Deluxe)
-  Food ordering system with automatic bill generation
-  Data persistence using file serialization
-  Thread-based data backup for efficiency
-  Exception handling for invalid and unavailable room cases

# Concepts Used
- OOP Principles: Inheritance, Encapsulation, Polymorphism  
- File Handling: Read/write data to maintain records  
- Serialization: Save and load customer data automatically  
- Multithreading: Background data backup using threads  
- Exception Handling: Custom exception (NotAvailable) for room availability
